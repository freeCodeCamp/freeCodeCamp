## تحويل مباشرة من ثنائي إلى سداسي عشري

يمكنك تقسيم الأرقام الثنائية الخاصة بك لتحويلها إلى أرقام سداسية عشرية.

### الطريقة العامة

1.  قطع سلسلة من الأرقام الثنائية الخاصة بك إلى مجموعات من أربعة ، بدءا من اليمين.
2.  إضافة أصفار إضافية إلى مقدمة الرقم الأول إذا لم يكن أربعة أرقام.
3.  قم بتحويل مجموعة مكونة من 4 أرقام في كل مرة.

يمكنك استخدام جدول التحويل هذا:

يمكن استخدام Colons لمحاذاة الأعمدة.

| ثنائي | عشري | | ------ | ----------: | | 0000 | 0 | | 0001 | 1 | | 0010 | 2 | | 0011 | 3 | | 0100 | 4 | | 0101 | 5 | | 0110 | 6 | | 0111 | 7 | | 1000 | 8 | | 1001 | 9 | | 1010 | أ | | 1011 | ب | | 1100 | ج | | 1101 | د | | 1110 | ه | | 1111 | F |

### مثال

تحويل الرقم الثنائي 01101000000001 إلى رقم سداسي عشري.

 `1. 01101000000001 => 01|1010|0000|0001 
 2. 01|1010|0000|0001 => 0001|1010|0000|0001 
 3. 0001|1010|0000|0001 => 1A01 
`
