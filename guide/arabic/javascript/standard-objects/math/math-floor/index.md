## الطابق الرياضيات

`Math.floor()` هي طريقة للكائن القياسي في الرياضيات تقريب رقم محدد لأسفل إلى العدد الصحيح التالي. `Math.floor()` أنه بالنسبة للأرقام السالبة فهذا يعني أن الرقم سيتم تقريبه "بعيداً عن 0" بدلاً من عدد القيمة المطلقة الأصغر حيث أن `Math.floor()` يعرض أكبر عدد صحيح أقل من أو يساوي الرقم المحدد.

### أمثلة

 `Math.floor(0.9)  //  0 
 Math.floor(1.3)  //  1 
 Math.floor(0.5)  //  0 
 Math.floor(-0.9) // -1 
 Math.floor(-1.3) // -2 
`
