## Date.prototype.getDate

ترجع الطريقة `getDate()` يوم الشهر للتاريخ المحدد وفقًا للوقت المحلي.

### بناء الجملة

 `dateObject.getDate() 
` 

### مثال

 `var Xmas95 = new Date('December 25, 1995 23:15:30'); 
 var day = Xmas95.getDate(); 
 
 console.log(day); // 25 
`
