## حالة Git

تعرض أوامر `git status` دليل العمل ومنطقة التدريج. يعرض المسارات التي تحتوي على اختلافات بين ملف `index` والتنفيذ الحالي لـ `HEAD` ، والمسارات التي تحتوي على اختلافات بين شجرة العمل وملف `index` ، والمسارات في شجرة العمل التي لا يتم تتبعها بواسطة Git (ولا يتم تجاهلها بواسطة [gitignore](https://git-scm.com/docs/gitignore)

لا يظهر لك الأمر `git status` أي معلومات تتعلق بتاريخ المشروع الملتزم. لهذا ، تحتاج إلى استخدام `git log` .

### استعمال

 `git status 
` 

قائمة الملفات التي يتم تنظيمها بدون ترتيب أو عدم تعقب.
