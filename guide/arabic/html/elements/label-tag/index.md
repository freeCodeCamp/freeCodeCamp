# علامة التصنيف

< **_label_** > يحدد تسمية لـ  نوع العنصر / العلامة.

### استعمال

 `<label for="id">Label</label> 
 <input type="text" name="text" id="id" value="yourvalue"><br> 
` 

كما ترون، ينبغي أن يكون _للسمة_ من العلامة يساوي السمة معرف العنصر ذات الصلة لربط معا.

### دعم النظام الأساسي

| متصفح | دعم عنصر | |: -----: |: -------------: | | انترنت اكسبلورر |: علامة _الاختيار_ البيضاء: | | موزيلا فايرفوكس |: علامة _الاختيار_ البيضاء: | | جوجل كروم |: علامة _الاختيار_ البيضاء: | | أوبرا |: علامة _الاختيار_ البيضاء: | سفاري |: علامة _الاختيار_ البيضاء: |

### سمات

| السمة | قيمة | الوصف | |: -------: |: ----: |: ---------: | | ل| _معرف_ العنصر _| يحدد أي عنصر نموذج يرتبط به ملصق | | form | form_ id | لتحديد _نموذج_ واحد أو أكثر ينتمي التسمية إلى |

### السمة العالمية

تدعم العلامة < **label** > السمات العامة في HTML.

### سمة الحدث

تدعم العلامة < **label** > سمات الأحداث في HTML.

> لا يتم تقديم العنصر < **label** > كأي شيء خاص للمستخدم. ومع ذلك ، فإنه يوفر تحسينًا قابلية الاستخدام لمستخدمي الماوس ، لأنه إذا قام المستخدم بالنقر فوق النص داخل العنصر ، فإنه يقوم بتحويل عنصر التحكم.

## ضع الكلمة المناسبة

تحدد العلامة `<label>` تسمية لعنصر `<input>` .

يمكن ربط التسمية بعنصر إما باستخدام السمة "for" أو بوضع العنصر داخل العنصر.

 `
<label for="peas">Do you like peas? 
  <input type="checkbox" name="peas" id="peas"> 
 </label> 
` 

 `
<label>Do you like peas? 
  <input type="checkbox" name="peas"> 
 </label> 
`
