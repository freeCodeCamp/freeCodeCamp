## قم بإنشاء كائن JavaScript أساسي

### طريقة:

تتشابه هذه المشكلة إلى حد ما مع المثال المعطى. إعطاء كائن `dog` خاصيتين جديدتين - `name` و `numLegs` - وتعيينها إلى سلسلة ورقم ، على التوالي.

### حل:

 `let dog = { 
  name: "George", 
  numLegs: 4 
 }; 
`
