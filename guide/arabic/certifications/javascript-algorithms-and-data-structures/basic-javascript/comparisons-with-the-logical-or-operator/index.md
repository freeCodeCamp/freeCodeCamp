## مقارنات مع المنطقي أو المشغل

### الدليل: اتبع الكود الموجود في المثال

**تنبيه المفسد**

أدناه هو نموذج التعليمة البرمجية (أبسط طريقة).

 `if ( val < 10 || val > 20 ) { 
    return "Outside"; 
 } 
` 

سيعود الرمز أعلاه "خارجي" فقط إذا كان `val` يتراوح بين 10 و 20 (ضمناً).
