## معالجة صفائف مع unshift ()

بينما `push()` إضافة عناصر إلى الجزء الخلفي من المصفوفة ، يضيف `unshift()` إلى الأمام. كل ما عليك القيام به هو:

 `var arr = [2, 3, 4, 5]; 
 arr.unshift(1); // Now, the array has 1 in the front 
`
