## استخدم المشغل الشرطي (الثالوثي)

### تلميح 1

استخدم المشغل الثلاثي للتحقق من المساواة.

### تحذير حل قدما!

 `function checkEqual(a, b) { 
  return (a = b ? true : false ); 
 } 
 
 checkEqual(1, 2); 
`
