## استخدم jQuery لتعديل الصفحة بأكملها

### شرح المشكلة

أضف الطبقات `animated` `hinge` إلى عنصر `body` .

#### روابط ذات صلة:

*   [.addClass ()](https://api.jquery.com/addClass/e)

### حل:

 `<script> 
  $("body").addClass("animated hinge"); 
  }); 
 </script> 
`
