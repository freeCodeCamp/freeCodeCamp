## استخدم خاصية الاختزال المرن

بينما يمكنك تعيين خصائص `flex-grow` و `flex-shrink` و `flex-basis` فردي. إذا احتجت لرؤية قيم كل منهم ، يمكنك القيام بذلك في سطر واحد باستخدام التنسيق التالي `flex: flex-grow flex-shrink flex-basis;` .

مثال:

 `#box-1 { 
    background-color: dodgerblue; 
    flex: 2 2 150px; 
    height: 200px; 
  } 
`
