## Vim-Plug

Gerenciador de plug-in Vim-Plug minimalista Vim.

### Prós:

*   Mais fácil de configurar: arquivo único. Não requer código clichê.
*   Mais fácil de usar: sintaxe concisa e intuitiva.
*   Instalação / atualização paralela super rápida (com qualquer um dos `+job` , `+python` , `+python3` , `+ruby` ou Neovim).
*   Cria clones superficiais para minimizar o uso de espaço em disco e o tempo de download.
*   Carregamento sob demanda para um tempo de inicialização mais rápido.
*   Pode revisar e reverter atualizações.
*   Suporte de ramificação / tag / confirmação.
*   Ganchos pós-atualização.
*   Suporte para plugins gerenciados externamente.

Você pode encontrar o guia de instalação [aqui](https://github.com/junegunn/vim-plug#installation) .

Alguns plugins úteis para você começar são:

*   [nerdtree](https://github.com/scrooloose/nerdtree) - Um explorador de sistema de arquivos para o Vim.
*   [Emmet-vim](https://github.com/mattn/emmet-vim/) - Suporte para expandir abreviações semelhantes a [emmet](https://emmet.io/) .
*   [vim-polygot](https://github.com/sheerun/vim-polyglot) - Um pacote de idiomas sólido para o Vim.
*   [Auto Pairs](https://github.com/jiangmiao/auto-pairs) - Insira ou exclua parênteses, parênteses, cotações em par.
*   [fugitive.vim](https://github.com/tpope/vim-fugitive) - Um wrapper git para o Vim.

Você pode adicionar mais plugins à sua instalação do Vim.
