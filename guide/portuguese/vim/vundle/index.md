## Vundle

Vundle é a abreviação de Vim bundle e é um gerenciador de plugins Vim.

Ele permite que você…

*   acompanhe e configure seus plugins diretamente no `~/.vimrc` (um [arquivo](https://stackoverflow.com/questions/164847/what-is-in-your-vimrc) para personalizar o Vim)
*   instalar plugins configurados (também conhecidos como scripts / bundle)
*   atualizar plugins configurados
*   pesquisar por nome todos os scripts disponíveis do Vim
*   limpar plugins não utilizados
*   execute as ações acima em um único pressionamento de tecla com o modo interativo

Para começar, siga [este guia](https://github.com/VundleVim/Vundle.Vim#quick-start) .

Alguns plugins úteis para você começar são:

*   [nerdtree](https://github.com/scrooloose/nerdtree) - Explorador de árvores de arquivos
*   [syntastic](https://github.com/vim-syntastic/syntastic) or [ale](https://github.com/w0rp/ale) - Verificador de sintaxe / linter
*   [tagbar](https://github.com/majutsushi/tagbar) - Exibe tags em uma janela
*   [ctrlp](https://github.com/kien/ctrlp.vim) - arquivo fuzzy, buffer, mru, tag, etc. localizador
*   [gitgutter](https://github.com/airblade/vim-gitgutter) - Mostra um diff de git na medianiz (coluna de sinal)
*   [fugitivo](https://github.com/tpope/vim-fugitive) - invólucro Git
*   [delimitMate](https://github.com/Raimondi/delimitMate) - Auto-preenchimento do modo de inserção para aspas, parênteses, colchetes, etc.
*   [tcomment](https://github.com/tomtom/tcomment_vim) - linhas de comentário qualquer que seja o idioma
*   [e mais toneladas!](https://vimawesome.com/)
