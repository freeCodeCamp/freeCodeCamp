## MongoDB

O MongoDB é um banco de dados não relacional de código aberto que usa uma estrutura semelhante a JSON para armazenar dados. Ele usa um modelo de documento para armazenar e recuperar os dados, em vez do modelo de tabela usado por bancos de dados sequenciais, como SQL ou Oracle.

### NOSQL Vs RDBMS

| Termos e conceitos do MongoDB | Termos e conceitos do SQL | | --- | --- | | Banco de dados | Banco de dados | | Colecção | Tabela | | documento | Row | | Field | Coluna | | Índice | Índice | | Documentos Embutidos | Junta de mesa |

### Características do MongoDB

1.  Banco de dados da próxima geração
2.  Nenhuma junção
3.  Clustering
4.  Código aberto
5.  Esquema menos
6.  Sem relacionamentos

### Recursos do MongoDB

1.  Banco de dados de documentos
2.  Alta performance
3.  Linguagem de consulta avançada
4.  Alta disponibilidade
5.  Escalabilidade Horizontal

### Database as a service (DBaaS)
Assim como diversos serviços, é possível criar uma base de dados na nuvem, ou seja, contratar algum serviço que lhe ofereça toda a interface de um banco de dados rodando em um servidor 24/7. Sendo essa questão seu principal atrativo, agilidade, facilidade e flexibilidade de manusear sem qualquer custo de hardware ou software em sua máquina.
