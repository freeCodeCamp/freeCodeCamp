## Use o analisador de corpo para analisar solicitações de POST

O analisador de corpo já deve ser adicionado ao seu projeto se você usou o clichê fornecido, mas se não deveria estar lá como:

```json
"dependencies": { 
    "body-parser": "^1.4.3", 
    ... 
```

Você precisa fazer para este desafio é passar o middleware para app.use (). Certifique-se de que vem antes dos caminhos que ele precisa ser usado.
