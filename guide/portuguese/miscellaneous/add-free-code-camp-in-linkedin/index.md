title: Add Free Code Camp in Linkedin
localeTitle: Adicione o Code Camp grátis no Linkedin
---
O LinkedIn reconhece o Free Code Camp como uma universidade.

Você pode explorar nossa atividade de ex-alunos [aqui](https://www.linkedin.com/edu/school?id=166029) .

Você deve adicionar o Free Code Camp ao seu perfil do LinkedIn. Veja como:

*   Siga este link: [Adicionar Free Code Camp](https://www.linkedin.com/profile/edit-education?school=Free+Code+Camp)
*   Siga estas instruções: ![](//discourse-user-assets.s3.amazonaws.com/original/2X/8/8b138a334f9cfc075fee7379b515a173a154934b.png)

Depois de concluir nosso currículo (ou concluir pelo menos um projeto sem fins lucrativos e, em seguida, obter um trabalho de codificação), emitiremos um certificado oficial de desenvolvimento completo para a Web que você também pode adicionar ao seu perfil do LinkedIn.
