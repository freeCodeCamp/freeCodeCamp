## Bulma

Bulma is an open source CSS framework based on flexbox and used by more than 100,000 developers.
Why Bulma?
* 100% Responsive.
* Modular.
* Modern.
* Free (Source code on Github).
* growing community.
* Easy to learn.
* Quick Customization.

Just one command to get started:
```terminal
$ npm install bulma
```

Read more about Bulma (here)[http://bulma.io/documentation/overview/start/]
