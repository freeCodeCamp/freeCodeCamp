# WordPress

WordPress是一个基于PHP和MySQL的免费开源内容管理系统, 功能包括插件架构和模板系统。它与博客最相关，但支持其他类型的网络内容，包括更传统的邮件列表和论坛，媒体画廊和在线商店。

WordPress占据了所有网站的近27％，目前在CMS市场份额中占主导地位。这个开源平台由庞大的社区提供支持，通过主题/插件和自定义软件为数十亿美元经济提供支持。
