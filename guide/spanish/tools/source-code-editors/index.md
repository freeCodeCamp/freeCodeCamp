## Editores de código fuente

Los editores de código fuente son los programas que permiten la creación de código y  
edición. Cualquier editor de texto puede ser usado para escribir código. Pero código dedicado  
Los editores ofrecen muchas funciones útiles, que incluyen: completar texto,  
coloreado de marcas, y pestañas / paneles para la edición en paralelo. Estas características pueden  
ser útil incluso para los nuevos usuarios que no conocen su camino alrededor de todo el  
opciones avanzadas.

Hay muchos, muchos editores disponibles para descargar o usar  
En línea, tanto de pago como de forma gratuita. Ofrecen muchas de las mismas características básicas, pero  
Cada uno tiene sus propias sutilezas. Más investigación puede ayudar a encontrar el que está  
bien por ti. Los siguientes editores son todos multiplataforma y  
Libre de usar o evaluar.

## [Soportes](http://brackets.io/) ![Icono de soportes](//discourse-user-assets.s3.amazonaws.com/original/2X/d/df30640741461569071875cf4ff6cad9924f8b32.png)

![Brackets Captura de pantalla](//discourse-user-assets.s3.amazonaws.com/original/2X/2/2f395a58d24ed551ed8440a10ae640777f1143b6.png)

Brackets es un editor de código abierto relativamente nuevo de Adobe. Es muy usuario

para encontrar e instalar a través del administrador de extensiones.

## [Código de Visual Studio](https://code.visualstudio.com/) ![Icono de código de Visual Studio](https://cdn-media-1.freecodecamp.org/imgr/b4vFsKa.png)

![Captura de pantalla de Visual Studio Code](https://code.visualstudio.com/home/home-screenshot-mac-lg.png)

Visual Studio Code es un editor de código fuente desarrollado por Microsoft en 2015, y ahora es uno de los editores de código más conocidos para el desarrollo web. Es compatible con muchos lenguajes de programación y, como otros, puede descargar complementos o extensiones para que sea más configurable para su estilo de codificación. Sus principales características son:

*   IntelliSense: para resaltado de sintaxis, autocompletar, definiciones de funciones y funciones inteligentes.
*   Depuración: ser capaz de depurar utilizando puntos de interrupción, pilas de llamadas y una consola interactiva.
*   Git incorporado: revise los archivos de diferencias, etapas, haga confirmaciones, presione y tire, todo disponible en el editor.
*   Extensiones: instale extensiones para agregar nuevos idiomas, temas, depuradores y conectar servicios adicionales.
