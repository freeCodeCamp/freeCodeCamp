# --description--

Para dar o efeito de uma cordilheira, adicione outra montanha, criando uma `div` imediatamente depois de `.left-mountain`. Dê à nova `div` uma `class` de `back-mountain`.

# --hints--

Você deve adicionar uma nova `div` dentro do `body`.

```js
assert.equal(document.querySelectorAll('body > div')?.length, 4);
```

Você deve dar à `div` uma `class` de `back-mountain`.

```js
assert.include(document.querySelector('div:not(.left-mountain, .ground, .penguin)')?.className, 'back-mountain');
```

Você deve colocar a `.back-mountain` após `.left-mountain`.

```js
assert.strictEqual(document.querySelector('.left-mountain')?.nextElementSibling, document.querySelector('.back-mountain'));
```
