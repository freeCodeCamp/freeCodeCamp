# --description--

Considere os divisores de 30: 1, 2, 3, 5, 6, 10, 15, 30.

Pode-se ver que, para cada divisor $d$ de 30, $d + \frac{30}{d}$ é primo.

Encontre a soma de todos os números inteiros positivos $n$ não excedendo $100.000.000$, tal que, para cada divisor $d$ de $n$, $d + \frac{n}{d}$ é um número primo.
