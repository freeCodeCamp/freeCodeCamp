# --description--

Mais uma coisa. Se você continuar clicando no botão `Load More Authors` até que não haja mais dados para carregar e que o texto se altere para `No more data to load`, o valor do cursor ainda será um `pointer`. Por que não mudar o valor do cursor para `not-allowed` em vez disso?

Acessar a propriedade `style` do botão `Load More Authors` e defina `cursor` como `not-allowed`.

Com isso, a página dos autores está completa!

const authorContainer = document.getElementById('author-container');
const loadMoreBtn = document.getElementById('load-more-btn');

let startingIndex = 0;
let endingIndex = 8;
let authorDataArr = [];

fetch('https://cdn.freecodecamp.org/curriculum/news-author-page/authors.json')

const fetchMoreAuthors = () => {
  startingIndex += 8;
  endingIndex += 8;

  displayAuthors(authorDataArr.slice(startingIndex, endingIndex));
  if (authorDataArr.length <= endingIndex) {
    loadMoreBtn.disabled = true;
