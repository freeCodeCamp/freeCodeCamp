# --description--
Одиниця `rem` означає корінь `em` та відноситься до розміру шрифту елемента `html`.

Оскільки елементи `label` є вбудованими за замовчуванням, всі вони зображаються пліч-о-пліч в одному рядку, що ускладнює читання тексту. Щоб вони зображалися в окремих рядках, додайте `display: block` до елемента `label` та додайте `margin` зі значенням `0.5rem 0`, щоб відокремити їх один від одного.

# --hints--

Ви повинні використати селектор `label`.

```js
assert.exists(new __helpers.CSSHelp(document).getStyle('label'));
```

Ви повинні додати властивість `display` зі значенням `block`.

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('label')?.display, 'block');
```

Ви повинні додати властивість `margin` зі значенням `0.5rem 0`.

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('label')?.margin, '0.5rem 0px');
```
