## Description
<section id='description'>
Если мы вычислим a2 mod 6 для 0 ≤ a ≤ 5, получим: 0,1,4,3,4,1. <p> Наибольшее значение a такое, что a2 ≡ a mod 6 равно 4. Назовем M (n) наибольшим значением a &lt;n таким, что a2 ≡ a (mod n). Итак, M (6) = 4. </p><p> Найти ΣM (n) для 1 ≤ n ≤ 107. </p>
