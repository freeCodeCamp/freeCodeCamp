## Description
<section id='description'>
Целочисленное разбиение числа n является способом записи n в виде суммы положительных целых чисел. <p> Разделы, которые отличаются только порядком их слагаемых, считаются одинаковыми. Разделение n на отдельные части представляет собой разбиение n, в котором каждая часть происходит не более одного раза. </p><p> Перегородки 5 на отдельные части: 5, 4 + 1 и 3 + 2. </p><p> Пусть f (n) - максимальное произведение частей любого такого разбиения n на отдельные части и пусть m (n) - число элементов любого такого разбиения n с этим произведением. </p><p> Таким образом, f (5) = 6 и m (5) = 2. </p><p> При n = 10 разбиение с наибольшим произведением составляет 10 = 2 + 3 + 5, что дает f (10) = 30 и m (10) = 3. И их произведение f (10) · m (10) = 30 · 3 = 90 </p><p> Можно проверить, что Σf (n) · m (n) для 1 ≤ n ≤ 100 = 1683550844462. </p><p> Найти Σf (n) · m (n) для 1 ≤ n ≤ 1014. Дайте свой ответ по модулю 982451653, 50-миллионный премьер. </p>
</section>
