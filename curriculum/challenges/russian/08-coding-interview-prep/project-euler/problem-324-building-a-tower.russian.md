## Description
<section id='description'>
Пусть f (n) представляет количество способов заполнения 3 × 3 × n башни блоками 2 × 1 × 1. Вам разрешено вращать блоки любым способом; однако повороты, отражения и т. д. самой башни считаются отличными. Например, (с q = 100000007): f (2) = 229, f (4) = 117805, f (10) mod q = 96149360, f (103) mod q = 24806056, f (106) mod q = 30808124. <p> Найдите f (1010000) mod 100000007. </p>
