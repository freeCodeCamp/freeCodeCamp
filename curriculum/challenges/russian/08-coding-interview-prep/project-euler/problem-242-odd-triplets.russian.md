## Description
<section id='description'>
Учитывая множество {1,2, ..., n}, определим f (n, k) как число его k-элементных подмножеств с нечетной суммой элементов. Например, f (5,3) = 4, так как множество {1,2,3,4,5} имеет четыре 3-элементарных подмножества, имеющих нечетную сумму элементов, т.е.: {1,2,4}, { 1,3,5}, {2,3,4} и {2,4,5}. <p> Когда все три значения n, k и f (n, k) нечетны, мы говорим, что они образуют нечетно-триплет [n, k, f (n, k)]. </p><p> Существует ровно пять нечетных триплетов с n ≤ 10, а именно: [1,1, f (1,1) = 1], [5,1, f (5,1) = 3], [5,5, f (5,5) = 1], [9,1, f (9,1) = 5] и [9,9, f (9,9) = 1]. </p><p> Сколько нечетных триплетов существует с n ≤ 1012? </p>
</section>
