## Description
<section id='description'>
Мы будем называть фракцию, которая не может быть отменена с помощью упругой дроби. Кроме того, мы будем определять устойчивость знаменателя R (d) как отношение его собственных дробей, которые являются упругими; например, R (12) = 4/11. <p> Устойчивость числа d&gt; 1 есть тогда φ (d) d - 1, где φ - функция тождества Эйлера. Далее мы определим coresilience числа n&gt; 1 как C (n) = n - φ (n) n - 1. Достаточность простого p равна C (p) = 1p - 1. Найдите сумму всех составных целых чисел 1 &lt;n ≤ 2 × 1011, для которого C (n) является единичной. </p>
</section>
