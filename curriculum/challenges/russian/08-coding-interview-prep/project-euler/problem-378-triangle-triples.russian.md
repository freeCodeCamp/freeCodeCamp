## Description
<section id='description'>
Пусть T (n) - n-е число треугольников, поэтому T (n) = <p> n (n + 1) 2 </p><p> , </p><p> Пусть dT (n) - число делителей T (n). Например: T (7) = 28 и dT (7) = 6. </p><p> Пусть Tr (n) - число троек (i, j, k) таких, что 1 ≤ i &lt;j &lt;k ≤ n и dT (i)&gt; dT (j)&gt; dT (k). Tr (20) = 14, Tr (100) = 5772 и Tr (1000) = 11174776. </p><p> Найдите Tr (60 000 000). Дайте последние 18 цифр вашего ответа. </p>
