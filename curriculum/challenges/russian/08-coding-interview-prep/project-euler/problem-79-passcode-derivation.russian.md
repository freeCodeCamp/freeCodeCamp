## Description
<section id='description'>
Общим методом защиты, используемым для онлайн-банкинга, является запрос пользователю о трех случайных символах из кода доступа. Например, если код доступа 531278, они могут запросить 2-й, 3-й и 5-й символы; ожидаемым ответом будет: 317. Текстовый файл keylog.txt содержит пятьдесят успешных попыток входа в систему. Учитывая, что три символа всегда запрашиваются по порядку, проанализируйте файл, чтобы определить максимально короткий секретный код неизвестной длины.
