## Description
<section id='description'>
Будем определять сбб как число вида p2q3, где p и q - различные простые числа. Например, 200 = 5223 или 120072949 = 232613. <p> Первые пять сплюсов - 72, 108, 200, 392 и 500. </p><p> Интересно, что 200 также является первым номером, для которого вы не можете изменить ни одну цифру, чтобы сделать премьер; мы будем называть такие числа первичными. Следующим простым доказательством, которое содержит смежную подстроку «200», является 1992008. </p><p> Найдите 200-й премьер-зуммер, содержащий непрерывную подстроку «200». </p>
