# --description--

4x4网格填充数字d，0≤d≤9。

可以看出，在网格中

6 3 3 0 5 0 4 3 0 7 1 4 1 2 4 5

每行和每列的总和值为12.此外，每个对角线的总和也是12。

在多少种方法中，您可以使用数字d，0≤d≤9填充4x4网格，以便每行，每列和两个对角线具有相同的总和？
