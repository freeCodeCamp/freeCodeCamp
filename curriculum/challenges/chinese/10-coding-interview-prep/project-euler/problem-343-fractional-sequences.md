# --description--

对于任何正整数k，分数xi / yi的有限序列ai由以下定义：a1 = 1 / k且ai =（xi-1 + 1）/（yi-1-1）减少到最低项i> 1 。当ai达到某个整数n时，序列停止。 （即，当yi = 1时。）定义f（k）= n。例如，对于k = 20：

1/20→2/19→3/18 = 1/6→2/5→3/4→4/3→5/2→6/1 = 6

所以f（20）= 6。

对于1≤k≤100，f（1）= 1，f（2）= 2，f（3）= 1且Σf（k3）= 118937。

找Σf（k3）为1≤k≤2×106。
