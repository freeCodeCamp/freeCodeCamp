# --description--

如果存在整数p，q，r，我们将调用正整数A和“亚历山大整数”，这样：

```
  A = p · q · r    and 1A 
```

=

```
  1p 
```

\+

```
  1q 
```

\+

```
  1r 
```

例如，630是亚历山大整数（p = 5，q = -7，r = -18）。实际上，630是第6个亚历山大整数，前6个亚历山大整数是：6,42,120,156,420和630。

找到第150000个亚历山大整数。
