# --description--

使用 `select` 元素给表单添加一个下拉列表很方便。 `select` 元素是包含一组 `option` 元素的容器，`option` 元素就是下拉列表的内容。 每一个元素都需要一个结束标签。

首先，在两个 `label` 元素下面添加一个 `select` 元素。 然后，在 `select` 元素内添加 5 个 `option` 元素。

# --hints--

应该在第三个 `fieldset` 内添加一个 `select` 元素。

```js
assert.exists(document.querySelector('fieldset:nth-child(3) > select'));
```

应该在 `select` 元素内添加 5 个 `option` 元素。

```js
assert.equal(document.querySelectorAll('fieldset:nth-child(3) > select > option')?.length, 5);
```
