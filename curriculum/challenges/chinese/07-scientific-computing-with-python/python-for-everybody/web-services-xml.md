# --question--

## --text--

以下 XML 有什么问题？

```xml
<person>
  <name>Chuck</name>
  <phone type="intl">
    +1 734 303 4456
  <email hide="yes" />
</person>
```

## --answers--

Email 标签缺少结束标签。

---

间距将导致 XML 无效。

---

Phone 标签缺少结束标签。

---

纯文本应使用 UTF-8 编码。
