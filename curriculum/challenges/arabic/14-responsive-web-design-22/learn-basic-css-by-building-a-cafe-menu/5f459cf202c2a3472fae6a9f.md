# --description--

لاحظ كيف يبدو سمك الخط كبير؟ القيمة الافتراضية لخاصية اسمها `border-width` هي `1px` لجميع حواف عناصر `hr`. عن طريق تغيير الحدود (borders) إلى نفس لون الخلفية، الارتفاع الإجمالي للخط هو `5px` (زائد عرض `3px` الحدود العلوية وزائد عرض `1px` السفلي).

غيّر خاصية `height` لعنصر `hr` إلى `2px`، بحيث يصبح الارتفاع الكلي له `4px`.

# --hints--

يجب عليك تعيين خاصية `height` إلى `2px`.

```js
const hasHeight = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style.height === '2px');
assert(hasHeight);
```

يجب أن يكون للعنصر `hr` الخاص بك `height` بقيمة `2px`.

```js
const hrHeight = new __helpers.CSSHelp(document).getStyle('hr')?.getPropertyValue('height');
assert(hrHeight === '2px');
```
