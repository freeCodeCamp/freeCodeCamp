# --description--

Puedes usar un elemento `hr` para mostrar una línea divisora entre secciones con diferente contenido.

Primero, añade un elemento `hr` entre el elemento `p` con clase `established` y el primer elemento `section`. Ten en cuenta que los elementos `hr` son de auto-cierre.

# --hints--

Debes añadir un elemento `hr`. Los elementos `hr` son de auto-cierre.

```js
assert(code.match(/<hr\s?\/?>/i));
assert(!code.match(/<\/hr>/i));
```

No debes cambiar el elemento `p` con clase `established` existente.

```js
assert($('p.established').length === 1);
```

No debes modificar el elemento `main` existente.

```js
assert($('main').length === 1);
```

El elemento `hr` debe estar entre el elemento `p` y el elemento `section`.

```js
assert($('hr')[0].previousElementSibling.tagName === 'P');
assert($('hr')[0].nextElementSibling.tagName === 'SECTION');
```
