## Description
<section id="description"> <strong>Objetivo:</strong> crear una aplicación <a href="https://codepen.io" target="_blank">CodePen.io</a> que sea funcionalmente similar a esta: <a href="https://codepen.io/freeCodeCamp/full/dNVazZ/" target="_blank">https://codepen.io/freeCodeCamp/full/dNVazZ/</a> . Cumplir las siguientes <a href="https://en.wikipedia.org/wiki/User_story" target="_blank">historias de usuario</a> . Utilice las bibliotecas o API que necesite. Dale tu propio estilo personal. <strong>Historia de usuario:</strong> Puedo crear recetas que tengan nombres e ingredientes. <strong>Historia de usuario:</strong> Puedo ver una vista de índice donde se pueden ver los nombres de todas las recetas. <strong>Historia de usuario:</strong> puedo hacer clic en cualquiera de esas recetas para verla. <strong>Historia de usuario:</strong> Puedo editar estas recetas. <strong>Historia de usuario:</strong> Puedo eliminar estas recetas. <strong>Historia de usuario:</strong> Todas las recetas nuevas que agrego se guardan en el almacenamiento local de mi navegador. Si actualizo la página, estas recetas todavía estarán allí. <strong>Sugerencia:</strong> debe prefijar sus claves de almacenamiento local en CodePen, es decir, <code>_username_recipes</code> Recuerde usar <a href="//github.com/FreeCodeCamp/freecodecamp/wiki/FreeCodeCamp-Get-Help" target="_blank">Read-Search-Ask</a> si se atasca. Cuando haya terminado, haga clic en el botón &quot;He completado este desafío&quot; e incluya un enlace a su CodePen. Puedes obtener comentarios sobre tu proyecto compartiéndolo con tus amigos en Facebook. </section>
