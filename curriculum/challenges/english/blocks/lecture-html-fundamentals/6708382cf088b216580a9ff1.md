# --interactive--

Forward propagation is the process of passing an input through a neural network to produce a prediction. It is the first half of every training step, and also what happens when you use a trained model to make predictions on new data.

### The Flow of Data

In forward propagation, data moves in one direction — from the input layer, through every hidden layer, to the output layer. At each layer, every neuron performs the same two operations:

1. Compute the **weighted sum**: multiply each incoming value by its weight, add them all up, then add the bias
2. Apply the **activation function** to the weighted sum

The output of one layer becomes the input to the next.

### A Step-by-Step Example

Consider a network with:
- 2 input neurons
- 1 hidden layer with 2 neurons (using ReLU)
- 1 output neuron (using Sigmoid)

**Input:**
```
x = [0.5, 0.8]
```

**Hidden layer weights and biases:**
```
w_hidden = [[0.4, 0.6],   # weights for neuron 1
            [0.3, 0.9]]   # weights for neuron 2
b_hidden = [0.1, 0.2]
```

**Step 1 — Compute weighted sums for hidden layer:**
```
h1_sum = (0.5 * 0.4) + (0.8 * 0.6) + 0.1 = 0.20 + 0.48 + 0.10 = 0.78
h2_sum = (0.5 * 0.3) + (0.8 * 0.9) + 0.2 = 0.15 + 0.72 + 0.20 = 1.07
```

**Step 2 — Apply ReLU:**
```
h1_out = relu(0.78) = 0.78
h2_out = relu(1.07) = 1.07
```

**Step 3 — Compute weighted sum for output neuron:**
```
w_output = [0.7, 0.5]
b_output = -0.3

out_sum = (0.78 * 0.7) + (1.07 * 0.5) + (-0.3) = 0.546 + 0.535 - 0.3 = 0.781
```

**Step 4 — Apply Sigmoid:**
```
prediction = sigmoid(0.781) ≈ 0.686
```

The network predicts approximately 0.686, which could be interpreted as "68.6% probability of the positive class."

### Forward Propagation in TensorFlow

You do not need to code the steps above manually. TensorFlow handles forward propagation automatically when you call `model.predict()` or pass data through the model:

```python
import tensorflow as tf
import numpy as np

model = tf.keras.Sequential([
    tf.keras.layers.Dense(2, activation='relu', input_shape=(2,)),
    tf.keras.layers.Dense(1, activation='sigmoid')
])

x = np.array([[0.5, 0.8]])
prediction = model.predict(x)
print(prediction)
```

### What Happens During Inference vs Training

Forward propagation occurs in both training and inference (using the model on new data). The difference is what happens after:

- **During training**: the prediction is compared to the correct label to compute loss, then backpropagation adjusts the weights
- **During inference**: the prediction is the final output — no loss is computed and no weights are changed

### Key Points

- Forward propagation passes input data through every layer from input to output
- Each neuron computes a weighted sum of its inputs, adds a bias, then applies an activation function
- The output of each layer becomes the input to the next
- Forward propagation is the same whether the model is training or making predictions on new data
