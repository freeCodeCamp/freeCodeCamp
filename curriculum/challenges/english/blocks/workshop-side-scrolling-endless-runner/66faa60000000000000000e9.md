# --description--
Birds fly at chest height. Crouch under them with the DOWN arrow. `setSize` shrinks the player's collision body; `setOffset` re-anchors it so the smaller body lines up with the lower half of the sprite.

The `update` line `this.applyDuck(this.duckKey.isDown);` is already wired below. Add the **closure and key** at the end of `create`:

```js
this.duckKey = this.input.keyboard.addKey("DOWN");
this.applyDuck = (down) => {
  if (down) {
    this.player.body.setSize(12, 8).setOffset(2, 8);
  } else {
    this.player.body.setSize(16, 16).setOffset(0, 0);
  }
};
```

The ducked body is `12 × 8` — slim enough to slide under the bird's `(24, 16)` body.
