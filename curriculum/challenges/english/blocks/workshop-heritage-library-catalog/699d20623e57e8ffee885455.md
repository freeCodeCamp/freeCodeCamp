# --description--

After splitting, each part still has extra whitespace. For example, splitting `"The Shining | King, Stephen"` produces `["The Shining ", " King, Stephen"]` - notice the spaces around the values. The `.trim()` method removes whitespace from both ends of a string.

To clean every element, you need to process each one individually. This is a perfect use case for a `for` loop.

Inside `parseCard`, declare an empty array named `trimmedParts`. Use a `for` loop to iterate over `parts`. On each iteration, push `parts[i].trim()` into `trimmedParts`. Then update the existing `return` statement to return `trimmedParts`.

After this step, the logged output should show clean values without extra spaces.
