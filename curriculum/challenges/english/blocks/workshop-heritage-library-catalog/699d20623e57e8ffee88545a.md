# --description--

With the catalog parsed, you can search through it by field. `findByAuthor` filters entries whose author field contains a search term.

Converting both the stored author and the search term to lowercase makes the search case-insensitive: `"King, Stephen".toLowerCase()` gives `"king, stephen"`, so searching for `"king"` still finds it. The `.includes()` method returns `true` if a string contains another string as a substring; `"king, stephen".includes("king")` returns `true`.

Create a function named `findByAuthor` that takes `catalog` and `author` as parameters. Declare `searchTerm` as `author.toLowerCase()`. Loop through `catalog`; for each entry, check whether `catalog[i].author.toLowerCase().includes(searchTerm)`. If it does, collect that entry. Return the results array.

function parseCard(rawString) {
  const parts = rawString.split("|");
  const trimmedParts = [];
  for (let i = 0; i < parts.length; i++) {
    trimmedParts.push(parts[i].trim());
  }
  const title = trimmedParts[0];
  const author = trimmedParts[1];
  const year = trimmedParts[2];
  const location = trimmedParts[3];
  return {
    title: title || "Unknown",
    author: author || "Unknown",
    year: year ? parseInt(year) : "Unknown",
    location: location || "Unknown"
  };
}

function parseCatalog(rawCards) {
  const catalog = [];
  for (let i = 0; i < rawCards.length; i++) {
    catalog.push(parseCard(rawCards[i]));
  }
  return catalog;
}
