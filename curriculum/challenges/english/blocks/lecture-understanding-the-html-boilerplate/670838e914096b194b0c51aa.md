# --interactive--

Before you can train your first neural network, you need three things: Python, TensorFlow, and a notebook environment. This lesson walks you through each one.

### Python

TensorFlow is a Python library, so Python is your starting point. Check whether Python is already installed by opening a terminal and running:

```bash
python --version
```

You should see something like `Python 3.10.12`. Any version from 3.8 to 3.11 works well with TensorFlow. If Python is not installed, download it from python.org.

### What is TensorFlow?

TensorFlow is an open-source deep learning library built by Google. It lets you define neural network architectures in a few lines of code, train models on CPU or GPU without changing your code, and export trained models for use on mobile, web, or server environments.

TensorFlow includes a high-level API called Keras that makes building models as simple as stacking layers. You will use Keras throughout this course.

### Installing TensorFlow

Install TensorFlow using pip, Python's package manager:

```bash
pip install tensorflow
```

To verify the installation worked, open Python and import TensorFlow:

```python
import tensorflow as tf
print(tf.__version__)
```

You should see a version number printed (e.g., `2.15.0`). If you see an error, make sure pip is up to date by running `pip install --upgrade pip` first.

### Jupyter Notebooks

Most deep learning work happens in Jupyter Notebooks — interactive documents that mix code, output, and text in the same file. They are ideal for learning because you can run one cell at a time and see results immediately.

```bash
pip install notebook
jupyter notebook
```

Your browser will open a file browser. Click New → Python 3 to create your first notebook.

### Google Colab (No Installation Required)

If you do not want to install anything locally, Google Colab is a free, browser-based notebook environment that comes with TensorFlow pre-installed and provides free access to a GPU. Visit colab.research.google.com and sign in with a Google account to start immediately.

### Your First TensorFlow Program

Once your environment is ready, run this code to confirm everything works:

```python
import tensorflow as tf

a = tf.constant([1.0, 2.0, 3.0])
b = tf.constant([4.0, 5.0, 6.0])
result = tf.multiply(a, b)

print("TensorFlow version:", tf.__version__)
print("Result:", result.numpy())
```

Expected output:

```
TensorFlow version: 2.15.0
Result: [ 4. 10. 18.]
```

### Key Points

- TensorFlow is a Python library for building and training neural networks
- Keras is TensorFlow's high-level API that makes defining models simple
- Install TensorFlow with `pip install tensorflow`
- Jupyter Notebooks let you run code interactively, one cell at a time
- Google Colab provides a free, GPU-enabled notebook environment with no local setup

Which command correctly installs TensorFlow using pip?

## --answers--

`python install tensorflow`

### --feedback--

`python install` is not a valid command. The correct tool for installing Python packages is `pip`.

---

`apt-get install tensorflow`

### --feedback--

`apt-get` is a Linux system package manager for operating system software, not Python libraries.

---

`pip install tensorflow`

---

`conda add tensorflow`

### --feedback--

The correct conda command would be `conda install tensorflow`, not `conda add`. But the standard cross-platform method is `pip install tensorflow`.
