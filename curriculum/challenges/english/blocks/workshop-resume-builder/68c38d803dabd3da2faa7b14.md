# --description--

In this workshop, you continue to practice working with TypeScript by building a Resume Builder app.

This app should help you to write and display your resume, and should contain sections like personal details, work experineces, skills, and education.

Lets start with writing ResumeBuilder component. This is a main component wihch should contain all the logic and a child components.

It should not get any props. But it should have a state typed
with all cv sections like: personalInfo, experiences, eductaions and skills
