# --description--

In this workshop, you will learn how to work with tsx by building a resume builder application. 

The first part of the application will include personal information like name, email, phone number, etc. 

Start by creating a `PersonalInfo` interface with the following properties:

`
fullName
email
phone
location
website
linkedin
summary
`
All of the properties should be a string type. The `linkedin` and `website` properties should be marked optional in the interface.
