# --description--

In prior lessons, you learned about the spread operator (`...`), introduced in ES6, it provides concise way to create shallow copies of arrays. Here's an example: 

```js
const originalArray = [1, 2, 3];
const copyArray = [...originalArray];

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```
