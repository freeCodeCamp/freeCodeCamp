# --description--

In prior lessons, you learned about the `slice()` method, which returns a new array containing a shallow copy of a portion of the original array, specified by start and end indices. Here's an example:

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.slice(1, 2);

console.log(newList); // ["Python"]
```

Use the `slice` method on `sample` to extract `windowSize` elements starting at `start`. Remember that `slice`'s second argument is an end index, not a length, so use `start + windowSize`. Return the result.
