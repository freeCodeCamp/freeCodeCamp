In prior lessons, you learned about the `indexOf()` method, which is used to find the first index of a specific element within an array. If the element cannot be found, it returns `-1`. Here's an example:

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

Call the `indexOf` method on `sample` with `value` as the argument. If `indexOf` returns `-1`, the value is not in the burst, so return the string `\"value does not exist\"`. Otherwise, return the index where the value was found.
