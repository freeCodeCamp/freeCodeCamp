# --description--

In Chinese, the number 80 is called `八十 (bā shí)`.

Numbers from 81 to 89 are formed by adding the unit digit (1–9) after `八十 (bā shí)` (80).

| Number | Chinese |
| :--- | :--- |
| 80 | `八十 (bā shí)` |
| 81 | `八十一 (bā shí yī)` |
| 82 | `八十二 (bā shí èr)` |
| 83 | `八十三 (bā shí sān)` |
| 84 | `八十四 (bā shí sì)` |
| 85 | `八十五 (bā shí wǔ)` |
| 86 | `八十六 (bā shí liù)` |
| 87 | `八十七 (bā shí qī)` |
| 88 | `八十八 (bā shí bā)` |
| 89 | `八十九 (bā shí jiǔ)` |
