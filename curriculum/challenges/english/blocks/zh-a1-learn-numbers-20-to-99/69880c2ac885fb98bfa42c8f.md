# --description--

Let's move on to the numbers in the 60s.

In Chinese, the number 60 is called `六十 (liù shí)`.

Numbers from 61 to 69 are formed by adding the Chinese for unit digit (1–9) after `六十 (liù shí)` (60).

| Number | Chinese |
| :--- | :--- |
| 60 | `六十 (liù shí)` |
| 61 | `六十一 (liù shí yī)` |
| 62 | `六十二 (liù shí èr)` |
| 63 | `六十三 (liù shí sān)` |
| 64 | `六十四 (liù shí sì)` |
| 65 | `六十五 (liù shí wǔ)` |
| 66 | `六十六 (liù shí liù)` |
| 67 | `六十七 (liù shí qī)` |
| 68 | `六十八 (liù shí bā)` |
| 69 | `六十九 (liù shí jiǔ)` |
