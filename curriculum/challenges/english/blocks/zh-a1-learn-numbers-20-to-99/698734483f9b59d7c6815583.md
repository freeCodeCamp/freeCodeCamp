### type 24 — Numbers (40–49)

# --description--

Now let’s move on to the numbers in the **40s**.

In Chinese, the number **40** is called `四十 (sì shí)`.

Numbers from **41 to 49** are formed by adding the unit digit (1–9) directly after  
`四十 (sì shí)` (40). The pattern is the same as with numbers in the 20s and 30s.

| Number | Chinese |
| :--- | :--- |
| 40 | `四十 (sì shí)` |
| 41 | `四十一 (sì shí yī)` |
| 42 | `四十二 (sì shí èr)` |
| 43 | `四十三 (sì shí sān)` |
| 44 | `四十四 (sì shí sì)` |
| 45 | `四十五 (sì shí wǔ)` |
| 46 | `四十六 (sì shí liù)` |
| 47 | `四十七 (sì shí qī)` |
| 48 | `四十八 (sì shí bā)` |
| 49 | `四十九 (sì shí jiǔ)` |
