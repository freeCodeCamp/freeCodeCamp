# --description--

Now it's time to learn the numbers in the 50s.

In Chinese, the number 50 is called `五十 (wǔ shí)`.

Numbers from 51 to 59 are formed by adding the unit digit (1–9) after `五十 (wǔ shí)` (50).

| Number | Chinese |
| :--- | :--- |
| 50 | `五十 (wǔ shí)` |
| 51 | `五十一 (wǔ shí yī)` |
| 52 | `五十二 (wǔ shí èr)` |
| 53 | `五十三 (wǔ shí sān)` |
| 54 | `五十四 (wǔ shí sì)` |
| 55 | `五十五 (wǔ shí wǔ)` |
| 56 | `五十六 (wǔ shí liù)` |
| 57 | `五十七 (wǔ shí qī)` |
| 58 | `五十八 (wǔ shí bā)` |
| 59 | `五十九 (wǔ shí jiǔ)` |
