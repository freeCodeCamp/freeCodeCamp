# --description--

In Chinese, the number 70 is called `七十 (qī shí)`.

Numbers from 71 to 79 are formed by adding the unit digit (1–9) after `七十 (qī shí)` (70).

| Number | Chinese |
| :--- | :--- |
| 70 | `七十 (qī shí)` |
| 71 | `七十一 (qī shí yī)` |
| 72 | `七十二 (qī shí èr)` |
| 73 | `七十三 (qī shí sān)` |
| 74 | `七十四 (qī shí sì)` |
| 75 | `七十五 (qī shí wǔ)` |
| 76 | `七十六 (qī shí liù)` |
| 77 | `七十七 (qī shí qī)` |
| 78 | `七十八 (qī shí bā)` |
| 79 | `七十九 (qī shí jiǔ)` |
