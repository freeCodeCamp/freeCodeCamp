# --description--

Let's practice writing some numbers that are between 50 and 59. To complete this task, keep in mind that in Chinese, the number 50 is called `五十 (wǔ shí)`.

The numbers from 0 to 9 are:

| Number | Chinese |
| :--- | :--- |
| 0 | `零 (líng)` |
| 1 | `一 (yī)` |
| 2 | `二 (èr)` |
| 3 | `三 (sān)` |
| 4 | `四 (sì)` |
| 5 | `五 (wǔ)` |
| 6 | `六 (liù)` |
| 7 | `七 (qī)` |
| 8 | `八 (bā)` |
| 9 | `九 (jiǔ)` |

These numbers are:
