# --description--

Now you know that the numbers from 11 to 19 are formed by the unit digit (1–9) after `十 (shí)` (10). The same logic applies to numbers from 21 to 29.

In Chinese, the number 20 is called `二十 (èr shí)`.

To form numbers from 21 to 29, you simply take `二十 (èr shí)` (20) and add the unit digit (1–9) directly after it.

| Number | Chinese |
| :--- | :--- |
| 20 | `二十 (èr shí)` |
| 21 | `二十一 (èr shí yī)` |
| 22 | `二十二 (èr shí èr)` |
| 23 | `二十三 (èr shí sān)` |
| 24 | `二十四 (èr shí sì)` |
| 25 | `二十五 (èr shí wǔ)` |
| 26 | `二十六 (èr shí liù)` |
| 27 | `二十七 (èr shí qī)` |
| 28 | `二十八 (èr shí bā)` |
| 29 | `二十九 (èr shí jiǔ)` |
