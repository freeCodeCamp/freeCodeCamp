# --explanation--

Ángela says `¿Navarro con dos erres?` This is a confirmation question.

`Con` introduces extra information about the spelling.

`Dos` is the number two.

`Erres` is the plural form of `erre`, the name of the letter `r`.

Ángela is confirming that the last name Navarro is written with two letters `r`.
