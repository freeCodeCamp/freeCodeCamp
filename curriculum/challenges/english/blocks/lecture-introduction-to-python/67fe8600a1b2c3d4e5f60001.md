# --description--

In the last lesson, you learned how to install Python on your machine. Now it's time to write your very first Python program and run it! This is an important step — once you know how to run code locally, you can experiment with everything you learn throughout this course.

## Creating Your First Python File

Python programs are saved as plain text files with a `.py` extension. You can create these files using any basic text editor (like Notepad on Windows or TextEdit on macOS), or a code editor like VS Code, which is a popular free choice among developers.

Follow these steps to write your first program:

**Step 1** — Open your text editor or code editor of choice.

**Step 2** — Type the following line of code:

```python
print("Hello, World!")
```

**Step 3** — Save the file with a name ending in `.py`. For example, save it as `hello.py`. Make a note of the folder where you save it.

## Running Your Python File

Now that you have your file ready, you need to use the terminal to run it.

A <dfn>terminal</dfn> (also called a command line or shell) lets you navigate your file system and run programs by typing commands. You already saw the terminal in the previous lesson when verifying your Python installation.

Here's how to run your `hello.py` file:

**Step 1** — Open your terminal:

- On **Windows**: open **PowerShell** or **Command Prompt** from the Start menu.
- On **macOS**: open the **Terminal** app (you can find it in Applications → Utilities).
- On **Linux**: open your system's terminal application.

**Step 2** — Navigate to the folder where you saved `hello.py` using the `cd` command. For example, if you saved the file on your Desktop, type:

```bash
cd Desktop
```

If your file is inside a folder called `my_projects` on your Desktop, you would type:

```bash
cd Desktop/my_projects
```

**Step 3** — Run the file by typing `python` (or `python3` on some systems) followed by the filename:

```bash
python hello.py
```

You should see the following output printed to the terminal:

```
Hello, World!
```

Congratulations — you just ran your first Python program!

## The Python Interactive Shell

Python also comes with an interactive shell (also called a REPL — Read, Evaluate, Print, Loop) that lets you type and run Python code one line at a time, without needing a file. You can open it by typing `python` (or `python3`) in your terminal with no filename:

```bash
python
```

You'll see a prompt that looks like this:

```
Python 3.x.x ...
Type "help", "copyright", "credits" or "license" for more information.
>>>
```

The `>>>` prompt means Python is ready for your input. Try typing the same line you wrote in your file:

```python
>>> print("Hello, World!")
Hello, World!
```

To exit the interactive shell, type `exit()` and press Enter, or press `Ctrl+D` on macOS/Linux, or `Ctrl+Z` then Enter on Windows.

## Experimenting With Code

Now that you know how to create `.py` files and run them, you are encouraged to try out the concepts you learn throughout this course in your own environment. The best way to learn programming is to practice — experiment with the examples, modify them, and see what happens!

For example, try changing the text inside the quotes in `hello.py` and running the file again:

```python
print("Python is awesome!")
```

The terminal will print whatever text you put in the quotes. Small experiments like this will help you build confidence and a deeper understanding of Python.
