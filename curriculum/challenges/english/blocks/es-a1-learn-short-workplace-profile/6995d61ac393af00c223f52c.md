# --explanation--

Sebastián says `¿Cómo se llama el ingeniero de software senior?`. This means "What's the name of the senior software engineer?"

When the word `Cómo` starts a question, it must carry an accent mark in the first letter `o`.

Also, remember that questions in Spanish start with an opening question mark, `¿`.
