# --description--

Write a function to return the factorial of a number.

Factorial of a number is given by:

<pre><big>n! = n * (n-1) * (n-2) * ..... * 1</big>
</pre>

For example:

<ul>
  <li><code>3! = 3 * 2 * 1 = 6</code></li>
  <li><code>4! = 4 * 3 * 2 * 1 = 24</code></li>
</ul>

**Note:** `0! = 1`

# --solutions--

```js
function factorial(n) {
  let sum = 1;
  while (n > 1) {
    sum *= n;
    n--;
  }
  return sum;
}
```
