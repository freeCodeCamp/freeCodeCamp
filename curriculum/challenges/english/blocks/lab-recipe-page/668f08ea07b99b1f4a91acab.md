# --interactive--

The perceptron is the simplest possible neural network — a single artificial neuron. Understanding how it works gives you the foundation for every more complex architecture in this course.

### What is a Perceptron?

In 1958, Frank Rosenblatt invented the perceptron, a mathematical model of a neuron that could learn from examples. The perceptron takes several numbers as input, combines them, and outputs a single number.

Here is the computation a perceptron performs:

```
output = activation(x1*w1 + x2*w2 + x3*w3 + bias)
```

Where:
- `x1, x2, x3` are the input values
- `w1, w2, w3` are the weights
- `bias` is an extra adjustable value
- `activation` is a function applied to the result

### Weights

A weight controls how much influence a particular input has on the output. A large positive weight means "this input strongly pushes the output up." A large negative weight means "this input strongly pushes the output down." A weight near zero means the input has little effect.

During training, the network adjusts its weights to make better predictions. This is the core of learning — finding the right set of weights.

### Bias

The bias is an extra learnable value added to the weighted sum before the activation function. It shifts the output up or down regardless of the inputs.

A good analogy is the y-intercept in the equation of a line (`y = mx + b`). The weight is like the slope `m` — it controls how the output responds to input changes. The bias is like `b` — it shifts the whole line up or down.

Without a bias, every neuron's output would be zero when all inputs are zero. The bias gives the neuron the flexibility to fire even when all inputs are zero, which is essential for learning many real-world patterns.

### A Concrete Example

Suppose you want to predict whether it will rain based on two inputs: cloud cover (0 to 1) and humidity (0 to 1).

```
cloud_cover = 0.8
humidity = 0.6
weight_cloud = 0.7
weight_humidity = 0.5
bias = -0.4

weighted_sum = (0.8 * 0.7) + (0.6 * 0.5) + (-0.4)
             = 0.56 + 0.30 - 0.40
             = 0.46
```

The result `0.46` is then passed through an activation function to produce the final output.

### Parameters

Weights and biases together are called the **parameters** of a neural network. Training is the process of finding the parameter values that minimise prediction error. A small neural network might have hundreds of parameters; a large language model like GPT-4 has hundreds of billions.

### Key Points

- A perceptron multiplies each input by a weight, sums the results, adds a bias, and passes the total through an activation function
- Weights control how strongly each input influences the output
- The bias shifts the output and allows the neuron to fire even when all inputs are zero
- Weights and biases together are called the network's parameters
- Training is the process of finding parameter values that minimise prediction error
