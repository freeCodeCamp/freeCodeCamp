# --explanation--

In Spanish, the verb `gustar` ("to like") is used to express that someone likes something. 

- `Me gusta crear` - I like to create.

To say that someone **doesn't** like something, you must write the word `no` at the start. For example:

- `No me gusta crear` - I don't like to create.

Esteban says `Le gusta crear ideas visuales`, so Diego likes to create visual ideas. 

The negative version would be `No le gusta crear ideas visuales.`
