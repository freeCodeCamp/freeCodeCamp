# --description--

The role of an engineer is very common in professional contexts. In Spanish, the word "engineer" changes based on gender. `Ingeniero` is the male form, while `ingeniera` is the female form. 

To specify the field, you can use this structure: 

**`Ingeniero(a)` + `de` + field**. 

Some engineering fields omit the word `de`.

For example:

- `Ingeniero de software` - Software engineer.

- `Ingeniero civil` - Civil engineer.
