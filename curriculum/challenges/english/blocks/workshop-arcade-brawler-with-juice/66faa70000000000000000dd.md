# --description--
Wire a Game Over scene that pops up when `playerHp <= 0`, shows the score + best, and offers a restart button.

Inside the player-damage callback, replace the `delayedCall(800, () => { this.invulnerable = false; ... })` reset with a fatal branch:

```js
if (this.playerHp <= 0) {
  this.player.active = false;
  this.player.body.setVelocity(0, 0);
  this.time.delayedCall(400, () => {
    this.scene.pause("MainScene");
    this.scene.launch("GameOverScene", {
      score: this.score,
      best: this.bestScore
    });
  });
  return;
}
this.time.delayedCall(800, () => {
  this.invulnerable = false;
  this.player.alpha = 1;
});
```

Then add a `GameOverScene` class between `UIScene` and `const config`, and update the config's scene list to `[MainScene, UIScene, GameOverScene]`. The Game Over scene displays SCORE / BEST and a `PLAY AGAIN` button that restarts MainScene + UIScene.

We `pause` MainScene rather than `stop` so the camera stays frozen behind the overlay. The 400ms delay before launching the scene lets the death blink animation finish.

class UIScene extends Phaser.Scene {
  constructor() {
    super({ key: "UIScene", active: true });
  }

  create() {
    this.scoreText = this.add
      .text(8, 8, "SCORE 0", { fontSize: "20px", color: "#ffffff" })
      .setScrollFactor(0);
    this.comboText = this.add
      .text(8, 36, "", { fontSize: "16px", color: "#ffeeaa" })
      .setScrollFactor(0);
  }

  update() {
    const main = this.scene.get("MainScene");
    if (!main) return;
    this.scoreText.setText(`SCORE ${main.score ?? 0}`);
    const combo = main.combo ?? 0;
    if (combo > 1) {
      this.comboText.setText(`COMBO x${combo}`);
      this.comboText.setScale(1 + combo * 0.05);
      this.comboText.setTint(
        Phaser.Display.Color.GetColor(255, Math.max(80, 255 - combo * 12), 100)
      );
    } else {
      this.comboText.setText("");
    }
  }
}
