# --explanation--

Chen Na is saying `加拿大人 (jiā ná dà rén)`, which means Canadian.

It combines `加拿大 (jiā ná dá)`, which means Canada, and `人 (rén)`, which means people. For example:

`他是加拿大人 (tā shì jiā ná dà rén)` — He is Canadian.
