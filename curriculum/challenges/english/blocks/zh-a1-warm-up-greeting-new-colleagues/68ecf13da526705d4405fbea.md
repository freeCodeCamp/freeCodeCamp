# --explanation--

Liu Ming is saying `新加坡人 (xīn jiā pō rén)`, which means Singaporean.

It combines `新加坡 (xīn jiā pō)`, which means Singapore, and `人 (rén)`, which means people. For example:

`她是新加坡人 (tā shì xīn jiā pō rén)` - She is Singaporean.
