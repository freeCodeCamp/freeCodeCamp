# --description--

Falsy values are values that evaluates to `false` in a boolean context. Examples of falsy values are:

- `""` (empty string)
- `0`
- `false`
- `null`
- `undefined`
- `NaN`

Change the assignment of `truthyOrFalsy` by setting its value to an empty string. After that, you'll see `false` logged to the console.
