# --description--

Lets dive into `flex` property, this sets how a flex item will grow or shrink to fit the space available in its flex container.

In our `style.css` file we have defined `flex: 2 2 10%` in this case the first number represents `flex-grow`, the second `flex-shrink` and the last one `flex-basis`. In the next section we are going to see examples for each one.
