# --description--

To begin working with the `flex` property, change the first number in `flex` from `0` to `1` in the `box` class of our `styles.css` file.

As we mentioned earlier, the first value represents `flex-grow`. This property controls how much extra space the flex item should take up if there is free space available in the container.

# --hints--

Your class `box` should have a property `flex` replace the first number `0` with the value `1`.

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('.box')?.flex, '1 0 250px');
```
