# --description--

Now, you will learn about scale generator of d3 library.   
There several types of [scale generators](https://d3js.org/d3-scale) in D3 library. Here, you will focus in two scale generator: 

  1. scaleOrdinal()
  2. scaleLinear()

In this step, you will learn about `scaleOrdinal()`.

*scaleOrdinal()*

```js
const color = d3.scaleOrdinal(["color1", "color2", "color3"], ["red", "green", "blue"]);
color("color1") // "red"
```

All element first array `["color1", "color2", "color3"]` of ordinal scale get mapped with each element of second array `["red", "green", "blue"]`.   
Now, create a ordinal scale for type to color named as `color`. ["Mammal", "Bird", "Reptile"] -> ["red", "green", "blue"]
