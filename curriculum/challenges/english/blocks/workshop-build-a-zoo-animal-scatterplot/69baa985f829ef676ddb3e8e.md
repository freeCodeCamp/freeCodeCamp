# --description--

**Let's make me container responsive**  
In previous step-1, you have build a container. And you have noticed that was static container, now you will make it responsive for different `windowWidth`. For this just add a Event listener on the window, call a fuction that resize that container for each window resize, simple.

```js
const margin = { top: 60, right: 60, bottom: 50, left: 60 };
const maxHeight = 500;
const maxWidth = 800;

function resizeFunction() {
  const windowWidth = window.innerWidth;

  // width and height for the content inside container:  
  let width = Math.min(maxWidth, windowWidth * 0.9) - margin.left - margin.right;  
  let height = Math.min(maxHeight, window.innerHeight * 0.7) - margin.top - margin.bottom;

  // width and height for the container:  
  d3.select("container")
    .attr("width", width + margin.left + margin.right)
    .attr("height", height + margin.top + margin.bottom);
}

resizeFunction();
window.addEventListener("resize", resizeFunction);
```
