# --description--

**Let's draw a circle inside that svg and color it**  
So far, we have created a responsive `svg` container, Now, Let's draw red circle inside.

```js
// a svg container with fixed width and height
const svg_circle = d3
  .select("body")
  .append("svg")
  .attr("width", '800px')
  .attr("height", '500px')
  .style("border", '2px solid black'); 

// appends a circle element at (cx,cy):(0,0) left-top position 
svg_circle
  .append("circle")
  .attr("cx", 0)   // cx goes Left to Right
  .attr("cy", 0)   // cy goes Top to Buttom
  .attr("r", 50)
  .style("fill", "red");
```

As you have already created responsive `svg` container, just append the `circle` into that. You will see **1/4th** of a red circle, rest of the circle is outside of the circle.
Then you can play with different `(cx,cy)` location 😀.

It gives clear understanding of co-ordinate system of `svg`, which plays a key role in D3.js library.
