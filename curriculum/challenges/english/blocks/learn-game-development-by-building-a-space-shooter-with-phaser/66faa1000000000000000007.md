# --description--

The first config property is `type`, which selects the renderer. `Phaser.AUTO` lets Phaser pick WebGL when available and fall back to Canvas otherwise.

Inside `config`, add the property `type` with the value `Phaser.AUTO`.

# --hints--

Your `config` object should have `type` set to `Phaser.AUTO`.
