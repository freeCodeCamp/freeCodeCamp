# --description--

In this workshop, throughput refers to the total number of attendees processed by a gate across all ticks in a single simulation (morning or night). Your `initializeThroughput` will prepare an object to track how many attendees each gate processes.

To complete your `initializeThroughput` function:

- Create an empty object named `summary`.
- Loop through each gate in the input `gates` array. For each gate, add a property to `summary` where:
  - The key is the gate's `id`.
  - The value is `0`.
- Return the `summary` object.

Your `initializeThroughput` function should not mutate the source input array `gates`.
