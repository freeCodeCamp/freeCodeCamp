# Welcome to Deep Learning with TensorFlow

This course will take you from zero programming or math background to building real neural networks that can see images, understand text, and generate new content.

You do not need to have studied machine learning before. Every concept is introduced from scratch, with plain-language explanations and hands-on TensorFlow code.

## What you will build

By the end of this course you will have trained models that:

- Classify images using convolutional neural networks
- Predict future values in a time series using LSTMs
- Generate new images using a GAN
- Understand and summarize text using a transformer

## How this course is structured

The course is divided into six blocks. Each block introduces one major area of deep learning, starting with the big picture and ending with working code.

Block 1 lays the foundation — what AI actually is, where deep learning fits inside it, and how to set up the tools you will use throughout the course.
