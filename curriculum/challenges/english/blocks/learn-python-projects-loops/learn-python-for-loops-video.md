# --questions--

## --text--

Which of the following is the correct way to iterate over each letter in the string `"Giraffe Academy"`?

## --answers--

```python
for (letter) in "Giraffe Academy":
    print(letter)
```

---

```python
for key, letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter, value in "Giraffe Academy":
    print(letter)
```

3
