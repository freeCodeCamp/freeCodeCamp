# --description--

Finally, let's define what happens when the radio button is checked— meaning the user has selected it.

Select `.contact-method-radio-btn:checked::before`, because we’re styling the inner circle only when the radio button is selected. 
Set `transform` to `translate(3px, 3px) scale(1)` and `background-color` to `lightgreen`.

- `transform: translate(3px, 3px) scale(1)` moves the inner circle slightly and scales it up to full size `(scale(1))`, making it visible.

- `background-color: lightgreen` fills the circle with color to show that the option is selected.

Because we added a `transition` earlier, this change will animate smoothly — the circle will grow and fade in, giving a polished, interactive feel.

This technique lets us fully customize how radio buttons behave and look, instead of relying on the browser’s default styling.
