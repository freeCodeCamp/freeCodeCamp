# --description--

Now, let's create the inner circle that will appear when the radio button is checked. We'll use the `::before` pseudo-element for this.

A pseudo-element like :`:before` lets you insert extra content before the actual element — often used for decorative purposes.

Select `.contact-method-radio-btn::before`. Set `display` to `block`, `content` to `" "`, `width` to `10px`, `height` to `10px`, and `border-radius` to `50%`.

What they mean:

- `content: " "` creates an empty visual box. Without this line, the pseudo-element wouldn’t appear at all.
- `display: block` makes it behave like a box instead of inline text.
- `width` and `height` set its size to 10×10 pixels.
- `border-radius: 50%` makes it perfectly circular.
