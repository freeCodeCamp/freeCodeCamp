# --description--

Now that all of the tabs have been deselected and all of the panels are hidden, you need to set the clicked tab to be selected.

Use the `setAttribute` method on the `tab` to set the `aria-selected` attribute to `"true"`. 

# --hints--

When a `tab` is clicked, the `aria-selected` attribute of the clicked tab should be set to `"true"`.

```js
const secondTab = document.querySelector("#tab-saturn");
secondTab.click();
assert.equal(secondTab.getAttribute("aria-selected"), "true");
```
