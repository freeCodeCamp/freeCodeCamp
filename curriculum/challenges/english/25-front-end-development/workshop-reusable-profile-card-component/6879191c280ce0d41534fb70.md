# --description--

When using `map()` in React, each item you render needs a special identifier called a `key`. 

Add a `key` prop to the rendered item and set its value to `profile.id`. Remember to wrap `profile.id` in curly braces, since it's a JavaScript variable.

# --hints--

Your `div` element should include a `key` prop.

```js
assert.match(code, /<\s*div\s+key\s*=\s*\{[^}]*\}/);
```

The `key` prop should be set to `profile.id`.

```js
assert.match(code, /<\s*div\s+key\s*=\s*\{\s*profile\.id\s*\}/);
```
