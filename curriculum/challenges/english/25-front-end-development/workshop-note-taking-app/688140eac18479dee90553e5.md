# --description--

Since you are using another `div` element, you will need to make this more accessible for screen readers. 

In a prior lecture, you learned about the `aria-live` attribute which is used to create a live region on a page. This will notify screen reader users of dynamic content changes in the live region as they occur.

In your opening `#status` tag, add an `aria-live` attribute and set its value to `"polite"`. 

The `polite` value will tell screen readers to wait until any current announcement is finished or until the user stops typing before announcing the update.
