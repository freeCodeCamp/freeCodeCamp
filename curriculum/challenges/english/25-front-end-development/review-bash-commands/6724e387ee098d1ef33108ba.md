# --description--

Interact with the terminal

`pwd`

`echo`

`exit`

`<command> --help`

`clear`



Investigate the files

`ls`

`ls -l`

`ls -a` `ls --all`

`more <filename>`

`find`

`find -name <filename>`



Navigate 

`cd <foldername>`

`..` in paths (`cd ..`)

`.` in paths (`cd .`)




Create and delete

`mkdir <foldername>`

`touch <filename>`

`rm <filename>`

`cp <file> <destination>`

`cp -r <folder_to_copy> <name_of_copy>`

`mkdir <parent_folder>/<new_folder>`

`rmdir <directory_name>`

`rm -r <folder>`



Move and rename

`mv <filename> <new_filename>`

`mv <file> <destination>`



Save terminal output in a file

`echo text >> filename`
