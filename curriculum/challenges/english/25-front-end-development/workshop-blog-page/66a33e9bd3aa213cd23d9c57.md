# --description--

For the final step, you will need to add the phone number and email address for the blog author.

Inside the `address` element, add a paragraph element with the text of `Phone: 555-555-5555`.

Below that paragraph element, add another paragraph element with the text of `Email: fake@email.com`.

And with those changes, your blog page is now complete.

# --hints--

You should have two `p` elements inside your `address` element.

```js
const addressElement = document.querySelector('footer #contact address');

assert.lengthOf(pElements, 2);
```

Your first `p` element should have the text of `Phone: 555-555-5555`.

```js
const pElement = document.querySelector('#contact p:first-child');
assert.strictEqual(pElement?.innerText, 'Phone: 555-555-5555');
```

Your second `p` element should have the text of `Email: fake@email.com`.

```js

assert.strictEqual(pElement?.innerText, 'Email: fake@email.com');
```
