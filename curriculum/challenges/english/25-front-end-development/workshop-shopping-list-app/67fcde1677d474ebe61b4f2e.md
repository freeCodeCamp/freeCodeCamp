Then, add an `onChange` event handler to the input element. Pass it an arrow function that takes `e` as an argument, which is the event object. Inside the arrow function, call `setQuery` and pass it `e.target.value`. This will update the `query` state variable with the current value of the input field whenever the user types in it.
