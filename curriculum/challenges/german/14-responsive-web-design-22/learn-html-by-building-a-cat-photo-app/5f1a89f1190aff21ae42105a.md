# --description--

Wie bei Radio-Buttons sind die Formulardaten für ausgewählte Checkboxen die `name` / `value`-Attributpaare. While the `value` attribute is optional, it's best practice to include it with any checkboxes or radio buttons on the page.

Füge ein `value`-Attribut zu jeder Checkbox hinzu. Setze zur Bequemlichkeit das `value`-Attribut jeder Checkbox auf den gleichen Wert wie sein `id`-Attribut.
