# Keyboard Shortcuts Guide for freeCodeCamp Learning Platform

## Overview
This guide provides a comprehensive list of keyboard shortcuts available on the freeCodeCamp learning platform to enhance productivity and improve the coding learning experience.

## Code Editor Shortcuts

| Shortcut | Action | Platform |
|----------|--------|----------|
| `Ctrl + S` / `Cmd + S` | Save current work | All |
| `Ctrl + /` / `Cmd + /` | Toggle line comment | All |
| `Ctrl + Shift + /` / `Cmd + Shift + /` | Toggle block comment | All |
| `Ctrl + Z` / `Cmd + Z` | Undo | All |
| `Ctrl + Y` / `Cmd + Y` | Redo | All |
| `Ctrl + F` / `Cmd + F` | Find in file | All |
| `Ctrl + H` / `Cmd + H` | Find and Replace | All |
| `Tab` | Indent | All |
| `Shift + Tab` | Outdent | All |

## Navigation Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + K` / `Cmd + K` | Quick jump to lesson | 
| `Arrow Up/Down` | Navigate between lessons |
| `Home` | Jump to beginning of code |
| `End` | Jump to end of code |
| `Ctrl + Home` / `Cmd + Home` | Jump to start of file |
| `Ctrl + End` / `Cmd + End` | Jump to end of file |

## Testing Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + Enter` / `Cmd + Enter` | Run tests |
| `Ctrl + Shift + T` / `Cmd + Shift + T` | Run all tests |

## Accessibility Shortcuts

| Shortcut | Action |
|----------|--------|
| `Tab` | Navigate through focusable elements |
| `Shift + Tab` | Navigate backwards through focusable elements |
| `Enter` / `Space` | Activate focused button or link |
| `Alt + H` | Toggle screen reader announcements |

## Tips for Effective Keyboard Navigation

- **Reduce Mouse Usage**: Using keyboard shortcuts significantly reduces reliance on the mouse, improving workflow speed.
- **Accessibility**: Keyboard shortcuts are essential for users with mobility disabilities or those using screen readers.
- **Efficiency**: Learning and using shortcuts can save considerable time during coding practice.
- **Consistency**: Most shortcuts follow industry standards used in popular editors like VSCode and GitHub.

## Customizing Shortcuts

Users can customize keyboard shortcuts through the Settings panel. To access Settings:

1. Click on your profile icon in the top-right corner
2. Select "Settings"
3. Navigate to "Keyboard Shortcuts"
4. Click on the shortcut you want to customize
5. Press your desired key combination
6. Click "Save"
