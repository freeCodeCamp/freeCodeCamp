freeCodeCamp Open-Source Codebase and Curriculum

freeCodeCamp is a community-driven platform where people can learn programming for free in a supportive environment. It operates as a donor-funded 501(c)(3) nonprofit organization with the goal of helping millions of people—especially those with busy schedules—transition into careers in technology. To date, the community has supported more than 100,000 individuals in securing their first developer jobs.

The curriculum covers full-stack web development as well as machine learning topics. It is entirely self-paced and includes thousands of interactive coding challenges designed to help learners gradually build practical, job-ready skills.

Certifications

freeCodeCamp offers a variety of free certifications that collectively form its Full-Stack Developer curriculum. These include:

Responsive Web Design
JavaScript
Front-End Development Libraries
Python
Relational Databases
Back-End Development and APIs

Each certification consists of multiple learning components such as interactive lessons, workshops, coding labs, quizzes, and review materials. Learners must complete five required projects before becoming eligible to take the certification exam. Successfully passing the exam allows them to claim the certification.

In addition to technical certifications, freeCodeCamp provides language certifications based on internationally recognized proficiency levels. These include:

A2 English for Developers (Beta)
B1 English for Developers (Beta)
A1 Professional Spanish (Beta)
A1 Professional Chinese (Beta)

These language programs are structured into modules containing warm-ups, instructional content, practice exercises, review sections, and quizzes. Completing all quizzes is required before attempting the final certification exam.

Once earned, certifications remain valid indefinitely and can be shared on resumes or professional platforms like LinkedIn. Each certification is verifiable via a unique link. However, if a learner violates the Academic Honesty Policy—such as by submitting plagiarized work—their certification may be revoked and they may be banned from the platform.

To further support career preparation, freeCodeCamp also includes additional resources like Coding Interview Prep, Project Euler challenges, Rosetta Code exercises, and a remix of The Odin Project. There is also a free Foundational C# certification created in collaboration with Microsoft.

The Learning Platform

The codebase described in this repository powers the live freeCodeCamp platform.

Beyond the curriculum itself, the community provides several additional learning and support resources:

A discussion forum where users can ask questions, receive help, and get feedback on projects—often within a few hours
A YouTube channel offering full-length courses on topics like Python, SQL, Android development, and more
A technical publication featuring thousands of tutorials and articles covering programming, mathematics, and computer science
A Discord server where learners and developers can interact, collaborate, and socialize in real time

Users can join the community through the platform’s sign-in page.

Reporting Bugs and Issues

If you believe you have discovered a bug, you should first consult the official guide on how to report issues and follow the outlined steps. After confirming that the issue is new and reproducible—and ideally that others are experiencing it as well—you can submit a detailed report by opening a GitHub issue. Including clear steps, context, and relevant information helps maintainers diagnose and fix the problem more effectively.

Reporting Security Issues and Responsible Disclosure

The freeCodeCamp team values responsible disclosure of security vulnerabilities. If you identify a potential issue that could impact users or the platform’s integrity, you are encouraged to follow the official security policy and reporting procedures to ensure it is handled appropriately.

Contributing

freeCodeCamp is sustained by the efforts of thousands of volunteers worldwide. Contributions of all kinds—whether code, documentation, or other improvements—are welcomed. New contributors are encouraged to review and follow the official contribution guidelines to get started and become part of the community.
