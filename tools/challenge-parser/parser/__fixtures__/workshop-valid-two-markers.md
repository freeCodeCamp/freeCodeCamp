# --description--

Add a navigation element with an unordered list containing three list items with the text "Home", "About", and "Contact".

# --seed--

## --seed-contents--

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Workshop Challenge</title>
  </head>
  <body>
    <header>
--fcc-editable-region--
      <nav>
        <ul>
          <li>Home</li>
          <li>About</li>
          <li>Contact</li>
        </ul>
      </nav>
--fcc-editable-region--
    </header>
    <main>
      <h1>Welcome to our website</h1>
    </main>
  </body>
</html>
```
