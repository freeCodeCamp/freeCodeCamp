## --seed-contents--

In this step, you'll add a div element with the class "container" inside the body element.

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Workshop Challenge</title>
  </head>
  <body>
    <div class="container"></div>
  </body>
</html>
```
