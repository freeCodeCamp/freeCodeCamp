# --description--

Create a form with an input field and a submit button.

# --seed--

## --seed-contents--

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Workshop Challenge</title>
  </head>
  <body>
--fcc-editable-region--
    <form>
--fcc-editable-region--
      <input type="text" placeholder="Enter your name">
--fcc-editable-region--
      <button type="submit">Submit</button>
    </form>
  </body>
</html>
```
