# Contributing to freeCodeCamp

We welcome contributions from everyone! Here are some ways you can help:

## How to Contribute

1. Fork the repository
2. Clone your fork locally
3. Create a new branch for your changes
4. Make your changes
5. Commit and push your changes
6. Open a Pull Request

## Types of Contributions

- Fix typos or grammatical errors
- Improve documentation
- Report bugs
- Suggest new features

## Code of Conduct

Please read our [Code of Conduct](https://www.freecodecamp.org/news/code-of-conduct/) before contributing.

## Getting Help

Visit our [forum](https://forum.freecodecamp.org) if you need help.
